# O tema escolhido foi o Sistema Solar, que se encaixa bem nas três representações porque seus elementos podem ser organizados sob lógicas muito diferentes:

## Lista linear — os planetas em ordem de distância do Sol. A relação aqui é sequencial: cada planeta vem depois do anterior, sem ramificações. 

## Grafo — Sol, planetas, luas, asteroides e cometas conectados por relações como gravidade, órbita ou influência mútua.

## Hierarquia — o Sistema Solar se divide em categorias (Estrela, Planetas, Pequenos corpos), que se subdividem em subtipos (rochosos, gasosos, asteroides, cometas). A relação é de subordinação: cada nível pertence ao de cima. Útil para classificar.
